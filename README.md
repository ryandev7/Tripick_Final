## :pencil2:개요
- 여행을 준비하는 사람들에게 여행지의 단순 정보가 아닌 여행자들이 직접 경험한 여행코스를 참고하여 현장감 있는 정보를 제공

- 여행자들 간의 소통과 정보 공유를 위한 커뮤니티 공간을 만들어 색다른 여행을 만들어 나갈 수 있는 장을 제공
